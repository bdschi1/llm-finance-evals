# PR: Financial-Reasoning Agent Evaluation Template (awesome-agents)

Upstream repo: https://github.com/kyrolabs/awesome-agents  
Pull request: https://github.com/kyrolabs/awesome-agents/pull/55

Summary:
- Adds a link under **Testing and Evaluation** to a financial-reasoning
  evaluation template hosted in this repo.
- Template defines PM/analyst-grade tasks and rubrics for:
  - Earnings analysis using filings and buyside/sellside research,
  - Portfolio risk sanity checks,
  - “Risk-factor landmine” detection.

Relevance:
- Shows how these evals can be used as a shared resource by agent frameworks.
- Demonstrates domain-specific evaluation work relevant to
  Portfolio Managers / Analysts / AI-evaluator positions.
